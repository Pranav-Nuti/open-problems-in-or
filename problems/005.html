Problem #5

Convex Relaxation Tightness for Mixed-Integer Programming

Author Sanjeeb Dash, Oktay Günlük
Difficulty Hard
Chance of Being Open High

Preliminary Notation

Consider a mixed-integer linear program (MIP):

\[ \min \{ c^\top x : Ax \geq b,\, x_j \in \mathbb{Z} \text{ for } j \in I,\, x \geq 0 \} \]

Let \( P = \{ x : Ax \geq b, x \geq 0 \} \) be the linear relaxation, and \( P_I = \text{conv}\{ x \in P : x_j \in \mathbb{Z} \;\forall j \in I \} \) the integer hull. The integrality gap for a given \(c\) is: \[ \frac{\min\{ c^\top x : x \in P_I \}}{\min\{ c^\top x : x \in P \}} \]

A cutting plane is a linear inequality \( \alpha^\top x \geq \beta \) valid for \( P_I \) but not for \( P \). The closure \( P' \) after adding all such cuts is the first Chvátal-Gomory closure.

Problem Statement

Open Problem: For general MIPs, the integrality gap can be arbitrarily large. A fundamental question is: For which classes of MIPs does the linear programming relaxation (possibly augmented with a polynomial number of cutting planes) provide a constant-factor approximation?

More precisely:

  1. Characterize the family of MIPs for which \( P \) (or \( P' \)) approximates \( P_I \) within a constant factor for all objectives \(c\).
  2. For capacitated facility location, set cover, and other classic problems: how many rounds of the Sherali-Adams (or Lasserre) hierarchy are needed to achieve a constant integrality gap? Is a polynomial number of rounds sufficient?
  3. Is there a poly-time algorithm to separate over the split closure for general MIPs?
